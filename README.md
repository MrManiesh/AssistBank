# AssistBank 🏦

A modern, fast, and highly intuitive **Cash Denomination Calculator** for bankers, shopkeepers, and anyone handling cash. Built entirely offline with a beautiful, minimalist UI using **Kotlin** and **Jetpack Compose**. 

Designed to save you time when counting vast amounts of currency. Features real-time calculation, instant bundle conversion, and persistent offline history tracking so you never lose your totals!

### ✨ Features
- **Real-Time Calculation**: See your grand total calculate instantly as you type.
- **Bundle Conversion**: Automatically translates total notes into easy-to-read "bundles" (100 notes = 1 bundle).
- **Persistent History**: Your last 5 calculations are securely saved locally and can be reloaded at any time.
- **Minimalist Aesthetic**: A beautiful, custom color palette using soft borderless UI components.
- **Scalable Architecture**: Future-proof `navigation-compose` routing ready to house new banking tools!

### 💻 Tech Stack
- **Language**: Kotlin
- **UI Toolkit**: Jetpack Compose (Material Design 3)
- **Architecture**: MVVM (Model-View-ViewModel) + StateFlow
- **Local Storage**: Room Database (SQLite)
- **Navigation**: Jetpack Navigation Compose
